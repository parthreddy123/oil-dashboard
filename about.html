How the Oil Scenario Engine Was Built

A technical deep-dive into an AI-powered geopolitical intelligence platform for Indian refinery strategy

The Problem

Indian oil refiners face a complex decision environment: crude prices are driven by geopolitical events (Hormuz disruption, OPEC politics, regional conflict), and refining margins depend on a basket of 6+ product crack spreads across Singapore benchmarks.

This system was built to continuously ingest news, score it against geopolitical scenarios using an LLM, and compute probability-weighted expected values for Brent crude and refinery gross margins — deployed as a static page that auto-refreshes every 3 hours.

Data Ingestion (6 APIs + RSS)

9 Parallel Scrapers via ThreadPoolExecutor
  • EIA API v2 — Brent (RBRTE), WTI (RWTC), petrol, diesel, ATF, propane spot prices. USD/gallon × 42 = USD/bbl.
  • yfinance — Live Brent futures (BZ=F) for more current pricing than EIA's daily close.
  • OilPrice.com — Dubai/Oman crude benchmarks + RSS news feed.
  • ExchangeRate API — USD/INR for converting Indian prices.
  • Google News — 24 search queries covering oil, OPEC, Iran, India refineries, geopolitics.
  • 9 RSS Feeds — Reuters, Al Jazeera, Bloomberg, Rigzone, S&P Global, Hellenic Shipping, Middle East Eye.
  • PPAC (India Govt) — Refinery throughput/utilization from PDF reports.
  • OPEC MOMR — Monthly production data for 11 member countries.
  • Company Press Releases — IOCL, BPCL, HPCL, Reliance, ONGC.

Each scraper inherits from BaseScraper with built-in retry logic, response caching (by URL+params hash), and rate limiting. All 9 run in parallel using Python's ThreadPoolExecutor(max_workers=6).

The Scenario Engine (Core Intelligence)

The heart of the system. Five geopolitical scenarios, each with analyst-consensus price targets, preconditions, and invalidators.

Step-by-Step Scoring Algorithm

Phase 1 — Article Scoring (LLM)

Each unscored news article is sent to Claude Haiku 4.5 in batches of 10. The LLM rates every article against all 5 scenarios on a [-1.0, +1.0] scale, citing which precondition or invalidator is affected:

  • +0.3 to +0.6 — Confirms a precondition
  • -0.3 to -0.6 — Undermines a precondition
  • -0.7 to -1.0 — Matches an invalidator
  • -0.1 to +0.1 — Neutral

Result: 5 signals per article, stored in article_signals table.

Phase 2 — Softmax Probability Computation

Average signal per scenario is computed across all scored articles, then converted to probabilities via softmax:

score[s] = AVG(signal) for scenario s
weight[s] = exp(score[s] / T) / Σ exp(score[i] / T)
where T = 0.5 (temperature)

Lower temperature = more decisive separation between scenarios. Result: probabilities summing to 100%.

Phase 3 — Expected Value Computation

Probability-weighted expected values for each KPI across 3 horizons (1m, 3m, 6m):

EV(Brent) = Σ weight[s] × scenario_price[s]
EV(GRM) = Σ weight[s] × scenario_grm[s]
Phase 4 — Strategic Narrative (LLM)

Top 20 articles + probabilities are synthesized into a 3-bullet executive narrative by Claude Haiku, prompted with McKinsey senior partner tone: concise, specific, no hedging. Max 15 words per bullet.

Phase 5 — Per-Scenario Assessment

LLM writes 1 sentence per scenario explaining its probability, citing the specific signal or event driving the score.

Indian Refinery GRM Calculation

The Gross Refining Margin uses the Indian crude basket and Singapore-benchmarked product prices:

Indian Basket = 0.72 × Dubai/Oman + 0.28 × Brent

GRM = (Weighted Product Revenue) − Crude Cost
    = (0.42×Diesel + 0.22×Petrol + 0.12×Naphtha + 0.10×ATF + 0.08×FuelOil + 0.06×LPG) − Basket

Product prices are adjusted from US Gulf Coast to Singapore benchmarks:

  • Diesel: −$3/bbl, Petrol: −$2.5/bbl, ATF: −$2/bbl
  • Naphtha: estimated at 90% of petrol (no direct EIA series)
  • Fuel Oil: estimated at 65% of crude + $10 Singapore premium
  • LPG: +$5/bbl Singapore premium

CI/CD Pipeline

GitHub Actions — Every 3 Hours
  1. Checkout main branch
  2. Setup Python 3.12 with pip cache
  3. Run full pipeline — 9 parallel scrapers → impact tagging → scenario scoring → narrative generation → email digest
  4. Generate HTML via export_html.py — reads DB, renders Mermaid diagrams, outputs static report
  5. Deploy to gh-pages branch (force orphan = clean history each run)

Manual Refresh

The dashboard has a "Refresh Data" button that triggers the pipeline on-demand:

Button click → Cloudflare Worker (token proxy) → GitHub Actions repository_dispatch → Full pipeline → New HTML deployed

The page auto-polls with HEAD requests (ETag/Last-Modified) and reloads when new content is detected.

Database Schema

SQLite in WAL mode for concurrent reads during scraping. 11 tables:

  • crude_prices — Brent, WTI, Dubai, Indian Basket
  • product_prices — Diesel, petrol, ATF, LPG
  • crack_spreads — Product-crude differentials
  • refinery_data — Throughput, utilization
  • trade_flows — Import/export volumes
  • global_events — OPEC production, turnarounds
  • news_articles — 1000+ articles, sentiment tagged
  • article_signals — LLM scores per scenario
  • scenario_narratives — LLM-generated insights
  • scenario_accuracy — Prediction tracking
  • fx_rates — USD/INR exchange rates

Daily Email Digest

Automated daily email via Gmail SMTP with:

  • Strategic narrative (3 bullets)
  • Scenario probability table with deltas vs yesterday
  • Top 5 signal articles
  • Prediction accuracy (10-day rolling average error)

Key Design Decisions

  • Static HTML over server — GitHub Pages is free, fast, and globally cached. No server to maintain. The "backend" runs only during CI/CD.
  • Claude Haiku over GPT-4 — Faster, cheaper, sufficient for signal scoring. Full scenario context fits in one prompt.
  • Softmax over simple averaging — Creates decisive probability separation. Temperature=0.5 amplifies signal differences.
  • Preconditions + Invalidators — Gives the LLM structured criteria rather than open-ended scoring. Reduces hallucination.
  • 36-hour article window — Keeps signals fresh. Older articles are still in DB but don't dominate probabilities.
  • Analyst consensus as base prices — Scenario oil prices from GS, JPM, MS, Citi, WoodMac, Rystad (Mar 2026). LLM can adjust ±15% based on recent news.
  • 3 horizons (1m, 3m, 6m) — 1-month captures immediate shock pricing; 3-month for tactical planning; 6-month for strategic decisions.
